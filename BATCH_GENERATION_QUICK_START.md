## ğŸ› **TROUBLESHOOTING**

### **Issue: "Generation Failed"**
**Solution**: Check API keys in `.env.local`:
```
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AIzaSyD...
```

### **Issue: "Slow Generation"**
**Possible Causes**:
- Large batch size (20 ads)
- Network latency
- Gemini API rate limits

**Solutions**:
- Try smaller batch (5-10 ads)
- Switch to "Fast" mode
- Wait a moment and retry

### **Issue: "Export Failed"**
**Solution**: Check browser console for errors. Ensure:
- Images loaded successfully
- No CORS errors
- Enough disk space

### **Issue: "No A/B Pairs"**
**This is normal if**:
- Batch size < 5 ads
- All ads have very different CTR scores
- AI didn't find good test pairs
